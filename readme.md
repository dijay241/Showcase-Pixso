# Figma to PowerPoint Plugin

Плагин для экспорта фреймов из Figma в PowerPoint презентации.

## Установка

```sh
# Установка зависимостей
npm install

# Сборка плагина
npm run build
```

## Сборка

```sh
# Компиляция TypeScript
npm run build

# Очистка собранных файлов
npm run build:clean
```

## Использование

1. Создайте новый плагин в Figma, используя файл `manifest.json`
2. Загрузите файлы `code.js` и `ui.html`
3. Выберите фреймы в Figma и запустите плагин
4. Экспортированные слайды сохранятся как PowerPoint файл

## Структура проекта

- `code.ts` - основной код плагина (TypeScript)
- `ui.html` - минимальный HTML с встроенным JavaScript для экспорта
- `code.js` - скомпилированный основной код
- `manifest.json` - манифест плагина

## Особенности

- ✅ **PptxGenJS** - встроенная версия библиотеки (без внешних зависимостей)
- ✅ **TypeScript** - полная типизация основного кода
- ✅ **Минимальный UI** - без визуального интерфейса, только функциональность
- ✅ **Автоматический экспорт** - сразу создает PowerPoint файл
- ✅ **Безопасность** - соответствует Content Security Policy Figma
- ✅ Поддержка текста, изображений, векторной графики
- ✅ Сохранение скругленных углов
- ✅ Экспорт линий со стрелками
- ✅ Поддержка AutoLayout фреймов
- ✅ Автоматическое закрытие плагина после экспорта
